Raspberry Pi RemoteSwitch plugin for radio controlled power sockets

Requires a 433 MHz radio module connected to a Raspberry GPIO pin.
For more information please see e. g. https://www.raspberrypi.org/forums/viewtopic.php?f=37&t=66946

To compile this plugin on a Raspberry Pi you need to compile and install wiringPi.
This is also required to use the plugin.

If you want to cross-compile, you need to put a precompiled libwiringPi.so into the
following folder: ../../../../../libraries/rpi/wiringPi/lib
